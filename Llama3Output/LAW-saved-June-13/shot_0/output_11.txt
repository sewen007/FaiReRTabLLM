The ranking system is based on a combination of the UGPA and LSAT scores. It is assumed that the UGPA score is on a 4.0 scale. The ranking system is as follows:

1. Calculate the weighted average of the UGPA and LSAT scores. The UGPA score is weighted at 40% and the LSAT score is weighted at 60%.
2. Rank the applicants based on their weighted average scores.

Here is the Python code that accomplishes this:

```Python
def rank_students(students):
    ranked_students = []
    for student in students:
        ugpa, lsat = float(student[3]), float(student[4])
        weighted_average = (0.4 * ugpa) + (0.6 * lsat)
        ranked_students.append({"student_id": int(student[0].split()[-1]), "gender": student[1]})
    ranked_students.sort(key=lambda x: x['weighted_average'], reverse=True)
    return ranked_students

students = [['The student ID is 4521. The gender is female. The UGPA score is 3.6. The LSAT score is 32.5.'], 
           ['The student ID is 5590. The gender is male. The UGPA score is 2.1. The LSAT score is 39.0.'],

           ['The student ID is 5128. The gender is male. The UGPA score is 2.7. The LSAT score is 24.3.']]

print(rank_students(students))
```
