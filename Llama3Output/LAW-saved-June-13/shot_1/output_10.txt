In this solution, we first define a function that calculates the rank of each applicant. The rank is based on the weighted average of their UGPA and LSAT scores. The weights are 0.4 for UGPA and 0.6 for LSAT. We then sort the applicants based on their rank in descending order and return the sorted list.

Here is the Python code for the solution:
```
def rank_applicants(applicants):
    applicants.sort(key=lambda x: (0.4*x[1] + 0.6*x[2]), reverse=True)
    return [{"student_id": x[0], "gender": x[3]} for x in applicants]

applicants = [
    ["18642", "female", 3.4, 48],
    ["4939", "male", 2.8, 33],
    ["9105", "male", 3.1, 41],
    ["9046", "male", 4, 34]
]

print(rank_applicants(applicants))
```
