Here is the Python code that can solve this problem:

```Python
students = [
    {"student_id": 15728, "gender": "female", "ugpa": 3.1, "lsat": 36.0},
    {"student_id": 3056, "gender": "female", "ugpa": 3.1, "lsat": 40.0},
    {"student_id": 19317, "gender": "female", "ugpa": 3.2, "lsat": 40.0},
    {"student_id": 3919, "gender": "female", "ugpa": 3.2, "lsat": 36.5},
    {"student_id": 21759, "gender": "male", "ugpa": 3.2, "lsat": 32.0}
]

students.sort(key=lambda x: (x["ugpa"], -x["lsat"]))

result = [{"student_id": student["student_id"], "gender": student["gender"]} for student in students]

print(result)
```
