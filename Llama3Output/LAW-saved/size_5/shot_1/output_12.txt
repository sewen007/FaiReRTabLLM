Here is the python code to rank the applicants based on their UGPA and LSAT scores:

```Python
applicants = [
    {'student_id': 18642, 'gender': 'female', 'UGPA': 3.4, 'LSAT': 48},
    {'student_id': 4939, 'gender':'male', 'UGPA': 2.8, 'LSAT': 33},
    {'student_id': 9105, 'gender':'male', 'UGPA': 3.1, 'LSAT': 41},
    {'student_id': 9046, 'gender':'male', 'UGPA': 4, 'LSAT': 34}
]

# Define a function to rank applicants
def rank_applicants(applicants):
    ranked_applicants = sorted(applicants, key=lambda x: (x['UGPA'], x['LSAT']), reverse=True)
    return [{'student_id': applicant['student_id'], 'gender': applicant['gender']} for applicant in ranked_applicants]

# Rank the applicants
ranked_applicants = rank_applicants(applicants)

print(ranked_applicants)
```

In the above code, the `rank_applicants` function takes a list of applicants as input, sorts them based on their UGPA and LSAT scores, and returns a list of ranked applicants in the desired format. The `sorted` function is used with a lambda function as the key to specify the sorting criteria. The `reverse=True` argument is used to sort in descending order. The list comprehension is used to create the final output in the desired format. Finally, the `rank_applicants` function is called with the input applicants and the ranked applicants are printed.
