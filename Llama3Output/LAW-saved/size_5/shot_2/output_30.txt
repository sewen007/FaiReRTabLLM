This is a Python script that solves the problem:
```python
import json

# Input data
data = [
    {"student_id": 6749, "gender": "female", "ugpa": 3.6, "lsat": 32.0},
    {"student_id": 1005, "gender": "male", "ugpa": 3.7, "lsat": 35.0},
    {"student_id": 14939, "gender": "male", "ugpa": 3.2, "lsat": 38.0},
    {"student_id": 10190, "gender": "male", "ugpa": 3.3, "lsat": 36.0},
    {"student_id": 16014, "gender": "male", "ugpa": 3.3, "lsat": 35.0}
]

# Sort the data based on ugpa and lsat scores
sorted_data = sorted(data, key=lambda x: (-x['ugpa'], -x['lsat']))

# Create the output JSON
output = [{"student_id": x["student_id"], "gender": x["gender"]} for x in sorted_data]

# Print the output JSON
print(json.dumps(output))
```
