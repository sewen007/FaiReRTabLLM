Solution:

Here is a Python script that will rank the applicants based on their UGPA and LSAT scores. It uses a simple weighted sum to calculate the total score for each applicant, with UGPA weighted at 0.4 and LSAT weighted at 0.6.

```Python
def rank_applicants(applicants):
    ranked_applicants = []
    for applicant in applicants:
        student_id = int(applicant[0].split()[-1])
        gender = applicant[0].split()[1]
        ugpa = float(applicant[2])
        lsat = float(applicant[4])
        total_score = 0.4 * ugpa + 0.6 * lsat
        ranked_applicants.append({"student_id": student_id, "gender": gender})
    ranked_applicants.sort(key=lambda x: (-x['total_score'], x['student_id']))
    return ranked_applicants

applicants = [['The student ID is 13817. The gender is female. The UGPA score is 3.5. The LSAT score is 36.0.'],
              ['The student ID is 14385. The gender is male. The UGPA score is 3.6. The LSAT score is 34.0.'],
              ['The student ID is 13891. The gender is female. The UGPA score is 3.5. The LSAT score is 43.0.']]
print(rank_applicants(applicants))
```
