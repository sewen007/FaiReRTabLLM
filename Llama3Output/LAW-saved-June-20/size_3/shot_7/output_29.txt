Solution:
Here is a Python solution that uses a list comprehension and the built-in `sorted` function to rank the applicants:
```
applicants = [
    {'student_id': 10197, 'gender': 'female', 'ugpa': 2.5, 'lsat': 36.0},
    {'student_id': 14589, 'gender': 'female', 'ugpa': 3.3, 'lsat': 36.0},
    {'student_id': 4938, 'gender':'male', 'ugpa': 2.8, 'lsat': 39.0}
]

ranked_applicants = sorted(applicants, key=lambda x: (-x['lsat'], -x['ugpa']))

result = [{'student_id': a['student_id'], 'gender': a['gender']} for a in ranked_applicants]

print(result)
```
This code first defines the list of applicants as a list of dictionaries. Each dictionary represents an applicant and contains their student ID, gender, UGPA score, and LSAT score.

The `sorted` function is then used to sort the applicants based on their LSAT and UGPA scores in descending order. The `key` argument of the `sorted` function is a lambda function that takes a dictionary as input and returns a tuple containing the LSAT score and UGPA score in descending order.

Finally, a list comprehension is used to create a new list of dictionaries that contains only the student ID and gender for each applicant. This list is then printed as the result.
