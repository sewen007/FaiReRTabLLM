The ranking is based on a combination of the UGPA score and LSAT score. The applicants with the highest combined score are ranked first. The UGPA score is weighted at 0.4 and the LSAT score is weighted at 0.6. 

Here is a Python function to solve this problem:

```Python
def rank_students(applicants):
    applicants = [dict(zip(['student_id', 'gender', 'UGPA', 'LSAT'], [i[0], i[1], float(i[2]), float(i[3])])) for i in applicants]
    applicants.sort(key=lambda x: 0.4*x['UGPA'] + 0.6*x['LSAT'], reverse=True)
    return [{"student_id": i['student_id'], "gender": i['gender']} for i in applicants]
```

You can use this function to rank the applicants as follows:

```Python
applicants = [['The student ID is 17657. The gender is female. The UGPA score is 2.8. The LSAT score is 30.5.'], ['The student ID is 21394. The gender is female. The UGPA score is 3.3. The LSAT score is 43.0.'], ['The student ID is 360. The gender is female. The UGPA score is 2.7. The LSAT score is 27.0.']]
print(rank_students(applicants))
```
