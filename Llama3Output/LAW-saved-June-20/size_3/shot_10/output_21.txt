Here is the Python code that solves the problem:

```python
import json

def rank_applicants(applicants):
    ranked_applicants = sorted(applicants, key=lambda x: (-x[3], -x[2]))
    ranked_results = []
    for applicant in ranked_applicants:
        ranked_results.append({
            "student_id": applicant[0],
            "gender": applicant[1]
        })
    return ranked_results

applicants = [
    [21625, 'female', 3.1, 23.0],
    [14137, 'female', 2.5, 32.0],
    [19687,'male', 2.8, 41.0]
]

print(json.dumps(rank_applicants(applicants)))
```

This code sorts the applicants based on their UGPA and LSAT scores in descending order. It then creates a list of dictionaries where each dictionary represents an applicant with their ID and gender. The `json.dumps()` function is used to convert the list of dictionaries into a JSON string.
