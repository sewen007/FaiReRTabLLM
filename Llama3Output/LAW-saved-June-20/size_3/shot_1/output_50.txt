```Python
def rank_applicants(applicants):
    ranked_applicants = sorted(applicants, key=lambda x: (-x[3], -x[2]))
    result = []
    for applicant in ranked_applicants:
        result.append({"student_id": applicant[0], "gender": "male" if applicant[1] == "male" else "female"})
    return result

applicants = []
for applicant in [[19283, "male", 3.6, 38.0], [6126, "male", 3.6, 29.5], [18494, "female", 3.0, 30.0]]:
    applicants.append(applicant)

print(rank_applicants(applicants))
```

This code sorts the applicants based on their UGPA and LSAT scores in descending order. It then constructs the result list by iterating over the sorted applicants and adding the student ID and gender to each dictionary in the result list. The result is then printed.
